# WORKHIVE
WorkHive – Job Portal & Collaboration Platform
WorkHive is a full-stack web application designed to connect job seekers and recruiters through a modern and user-friendly platform. The application enables users to create accounts, manage profiles, search and apply for jobs, while recruiters can post job opportunities and manage applications. Built with React, Node.js, Express.js, and MongoDB, WorkHive provides secure authentication, role-based access control, responsive design, and real-time interaction features. The platform aims to simplify the hiring process and create an efficient workspace for both employers and candidates.

Features

Role-Based Access Control

Job Posting and Management

Job Search and Application System

Profile Management

Secure JWT Authentication

Responsive User Interface

WorkHive – Job Portal & Team Collaboration Platform
WorkHive is a full-stack web application that combines job recruitment and team collaboration in a single platform. It enables job seekers to discover and apply for opportunities, recruiters to post and manage jobs, and users to create collaborative teams for project-based work. The platform features secure authentication, role-based access control, job management, application tracking, team creation, group collaboration, and a responsive user interface. Built using React, Node.js, Express.js, and MongoDB, WorkHive aims to streamline hiring, networking, and teamwork for students, professionals, and organizations.
